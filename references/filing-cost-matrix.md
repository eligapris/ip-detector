# Filing & Maintenance Cost Matrix

This matrix governs Phase 5 (cost projection). The goal is to produce a defensible 10-year cash-flow projection for filing and maintaining the IP across the major jurisdictions.

All figures are in USD, rounded to nearest $500. Figures are based on official fee schedules as of 2024–2025 and standard attorney cost bands. Always verify current fees with the official source before producing the dossier — fee schedules change annually.

## Jurisdiction priority

Not every concept needs worldwide filing. Default jurisdiction set by concept type:

| Concept type | Default jurisdictions | Rationale |
|--------------|----------------------|-----------|
| Pure software algorithm | US, EU, CN | Top 3 software markets; JP/KR add cost without proportional value |
| Hardware / semiconductor | US, EU, CN, JP, KR | All major manufacturing hubs |
| Cryptography | US, EU, CN | Plus JP if for financial infrastructure |
| Biotech / pharma | US, EU, JP, CN | Major pharma markets; CN patent quality improving |
| Consumer UI / design | US, EU, CN, JP, KR | All major consumer markets |
| Business method | US only | Most permissive jurisdiction post-Alice; EU rejects most |

If the user has not specified jurisdictions, use the default set for the concept type.

## EPO (European Patent Office)

### Filing & prosecution (Year 1)
- EPO filing fee: €1,500 ≈ $1,650
- Search fee: €1,500 ≈ $1,650
- Examination fee: €2,300 ≈ $2,530
- Designation fee: €1,000 ≈ $1,100
- Grant fee: €1,030 ≈ $1,130
- Translation fees (English + one other): $2,000–$4,000
- Attorney (European patent attorney): €8,000–€20,000 (default $13,000)

**Year 1 total (default)**: ~$22,000

### Validation fees (Year 2–3)
Once granted by EPO, you must validate in each national jurisdiction you want protection in.
- Top validation countries: DE, FR, UK, NL, IT, ES
- Per-country validation: $500–$1,500 (fees + translation)
- Validate in 4 countries: ~$4,000

### National annuities (Years 2–10)
Each validated country charges annual renewal fees. Approximate totals:
- Germany: €2,500 (cumulative Years 2–10) ≈ $2,750
- France: €2,200 ≈ $2,420
- UK: £2,500 ≈ $3,180
- Netherlands: €2,800 ≈ $3,080
- Italy: €3,200 ≈ $3,520
- Spain: €3,000 ≈ $3,300

**National annuities total (4 countries, Years 2–10)**: ~$12,000

### 10-year total cost of ownership (EPO + 4 validations)
~$38,000

## Copyright (no filing required, but registration recommended)

Copyright vests automatically on fixation. Registration is recommended for enforcement in the US:
- US Copyright Office registration: $65 (single work, online)
- Attorney fee for registration: $300–$1,000
- **Total cost (one-time)**: ~$1,000

No maintenance fees. Enforcement is by litigation, cost $500k–$2M per case (cheaper than patent litigation).

## Trademark (US, federal registration)

- USPTO filing fee (TEAS Plus): $250 per class
- Attorney fee: $500–$2,000 per class
- 10-year maintenance: §8 declaration + §9 renewal, $525 + $300 = $825 per class + attorney
- **10-year total cost per class (US)**: ~$3,000

For international trademark, Madrid Protocol: base fee CHF 653 (~$750) + per-designation fees. Budget $5,000–$15,000 for a 5-country Madrid filing.

## Trade secret (no filing cost, but ongoing protection cost)

- No filing fees
- Cost is in maintaining secrecy: NDAs, access controls, employee training, periodic audits
- Estimated $5,000–$20,000/year for a small company, more for larger
- 10-year protection cost: $50,000–$200,000
- Litigation cost if misappropriated: $1M–$5M

## Computing the 10-year cash flow for the dossier

For each jurisdiction in the chosen set, compute:
1. Year 1 cost (filing + prosecution + attorney)
2. Years 2–10 annuity/maintenance cost
3. Sum

Then add coordination overhead (5–10% of total, for IP firm management across jurisdictions).

The dossier's "Cost" section presents this as a single table:

## Verification rule

Before producing the dossier, verify at least 2 of the fee figures against current official sources:
- USPTO: https://www.uspto.gov/learning-and-resources/fees-and-payment/uspto-fee-schedule
- EPO: https://www.epo.org/applying/fees.html
- CNIPA: https://english.cnipa.gov.cn/
- JPO: https://www.jpo.go.jp/e/
- KIPO: https://www.kipo.go.kr/en/

If a verified fee differs from this matrix by more than 20%, use the verified fee and flag the discrepancy in the dossier.

## What this matrix does NOT include

- Litigation cost (separate; only relevant if the IP is enforced). For US patent litigation, budget $3M–$5M per case through trial.
- Licensing program cost (negotiating, drafting, auditing licensees). Budget $50k–$500k/year for an active licensing program.
- Insurance (defensive patent insurance is rare but available). Not included by default.
- Translation cost for additional jurisdictions beyond defaults. Add $2k–$5k per additional language.

These are flagged in the dossier's "Cost" section as "not included in 10-year projection" so the user is not surprised.
